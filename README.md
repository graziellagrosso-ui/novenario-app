# Novenário App (Vite + React + Tailwind + PWA)

## Rodar local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
- Crie um repositório no GitHub e suba os arquivos.
- Na Vercel: Add New Project → Import Git Repository → Framework: Vite → Build: `npm run build` → Output: `dist` → Deploy.
